# Linux Basics

## Common Commands
- `ls` : list files and directories
- `cd` : change directory
- `pwd` : show current directory
- `cp` : copy files
- `mv` : move or rename files
- `rm` : remove files or directories

## File Permissions
- `r` : read
- `w` : write
- `x` : execute
- Permissions are shown as rwx for owner, group and others

## Permission Example
- `chmod 755 file` gives full permissions to the owner and read/execute to others

## Users
- Linux is a multi-user operating system
- Each user has different permissions

## Why Linux Matters for Security
- Most servers run Linux
- Many security tools are Linux-based
